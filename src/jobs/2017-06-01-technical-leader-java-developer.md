## REQUISITI FONDAMENTALI:

- Laurea o Diploma in informatica oequivalenti;
- **7-8 anni** di esperienza come **sviluppatore Java**
- **Fluent English**
- **Java EE and Java SE 8**
- Enterprise design patterns
